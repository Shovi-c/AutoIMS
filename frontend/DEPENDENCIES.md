# Frontend Dependencies

## Production Dependencies

| Package             | Version  | Description                                |
| ------------------- | -------- | ------------------------------------------ |
| `react`             | ^19.2.0  | React library for building user interfaces |
| `react-dom`         | ^19.2.0  | React DOM rendering                        |
| `react-router-dom`  | ^7.12.0  | Declarative routing for React applications |
| `tailwindcss`       | ^4.1.18  | Utility-first CSS framework                |
| `@tailwindcss/vite` | ^4.1.18  | Vite plugin for Tailwind CSS               |
| `lucide-react`      | ^0.563.0 | Beautiful & consistent icon toolkit        |

## Installation

```bash
cd frontend
npm install
```

## Scripts

| Command           | Description                     |
| ----------------- | ------------------------------- |
| `npm run dev`     | Start development server (Vite) |
| `npm run build`   | Build for production            |
| `npm run preview` | Preview production build        |
| `npm run lint`    | Run ESLint                      |
